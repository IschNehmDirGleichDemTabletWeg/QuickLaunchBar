# Changelog

## [1.0] - 2026-03-06

### Initial release

- Quick Launch folder icon grid
- System tray with left-click show/hide
- Auto-hide on focus loss
- Dark theme, borderless draggable window
- Multi-monitor support (appears on active monitor)
- Settings dialog for columns/rows (stored in registry)
- Supports .lnk, .rdp, .exe, .url shortcuts
